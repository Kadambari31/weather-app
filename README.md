# Weather App

This is a simple Python Weather Application that fetches weather data using an API and displays it using a dashboard.

## Features
- Fetch real-time weather data
- Display weather information
- Simple Python implementation

## Technologies Used
- Python
- Weather API

## Files
- dashboard.py : Displays the weather dashboard
- weather_api.py : Handles API request

## How to Run
1. Install Python
2. Run the program:
   python dashboard.py
